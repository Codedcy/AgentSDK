# Agent SDK v0.1 R0 Release Harness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:test-driven-development for every behavior change and superpowers:verification-before-completion before checking off this plan. Use superpowers:executing-plans when running this plan in one session.

**Goal:** Establish a passing public-API release baseline and replace the active milestone order with the approved R0-R5 v0.1 track without deleting prior plans.

**Architecture:** Add one deterministic acceptance harness around the existing SDK facade and SQLite recovery path. Treat the existing roadmap as long-term backlog while a new release ledger becomes the operational source for v0.1.

**Tech Stack:** Python 3.13, asyncio, pytest, pytest-asyncio, Pydantic v2, SQLite/aiosqlite, existing AgentSDK public APIs.

## Global Constraints

- Do not add a new runtime capability in R0.
- The initial acceptance test must pass using existing behavior; do not commit skips or expected failures.
- Use public APIs in the acceptance test. Direct store access is allowed only for fixture cleanup or evidence assertions unavailable publicly.
- Use a temporary SQLite file and prove a second `AgentSDK` instance can reopen it.
- Session deletion must not delete workspace files.
- Preserve all old milestone documents; mark superseded ordering rather than rewriting their historical status.

            database_path=self.database_path,

        )

    def reopen(
        self,
        acompletion: Callable[..., Awaitable[object]] | None = None,
    ) -> AgentSDK:

            database_path=self.database_path,
            acompletion=acompletion or self.acompletion,
        )
```

The fixture uses a deterministic LiteLLM-shaped gateway already accepted by the
existing test utilities; it must not add a second provider abstraction.

@pytest.mark.asyncio
async def test_v01_release_baseline_reopens_and_deletes_history(
    v01_harness: V01Harness,
) -> None:
    workspace_file = v01_harness.workspace / "keep.txt"
    workspace_file.write_text("application-owned", encoding="utf-8")

    sdk = v01_harness.open()
    session = await sdk.sessions.create(
        workspaces=(v01_harness.workspace,),
        idempotency_key="v01-session",
    )
    agent = sdk.agents.define(
        AgentSpec(
        name="release-agent",
        model="test/model",
        )
    )
    handle = await sdk.runs.start(
        session.session_id,
        agent,
        "baseline",
        idempotency_key="v01-run",
    )
    terminal = await handle.result()
    assert terminal.output_text == "baseline complete"
    assert (await sdk.queries.timeline(handle.run_id)).run_id == handle.run_id
    await sdk.close()

    async def must_not_call(**_: object) -> object:
        raise AssertionError("completed Run must not call LiteLLM after reopen")

    reopened = v01_harness.reopen(must_not_call)
    observed = await reopened.queries.get_run(handle.run_id)
    assert observed.snapshot.status is RunStatus.COMPLETED
    await reopened.sessions.close(session.session_id)
    await reopened.sessions.delete(session.session_id)
    with pytest.raises(AgentSDKError) as deleted:
        await reopened.sessions.get(session.session_id)
    assert deleted.value.code is ErrorCode.NOT_FOUND
    assert workspace_file.read_text(encoding="utf-8") == "application-owned"
    await reopened.close()
```

Import `AgentSDK`, `AgentSDKError`, `AgentSpec`, `ErrorCode`, and `RunStatus` from
the package root so the test also checks the supported export surface.
