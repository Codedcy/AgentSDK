# Agent SDK 设计文档

本目录将架构决策拆为一篇概要设计和多篇独立特性设计。概要设计定义全局边界与不变量；特性设计定义状态、接口、事件、错误、配置和验收标准。

实施计划不与设计混写。书面设计获批后，在 `docs/plans/` 下生成：

- `00-roadmap.md`：全局路线图和依赖关系。
- `milestones/Mxx-*.md`：每个垂直切片或加固阶段一篇计划。
- `tasks/Mxx-Txxx-*.md`：每个可独立验收的任务一篇文档。

设计目录见[规格入口](../superpowers/specs/2026-07-13-agent-sdk-design.md)。
